# TCC – Agrupamento de Partidas de Futebol

Este repositório contém o código e os notebooks usados na etapa de formação dos vetores de atributos de partidas de futebol, para posterior aplicação de algoritmos de agrupamento.

> ⚠️ **IMPORTANTE:**  
> O arquivo `database.sqlite` **NÃO está neste repositório** (tamanho > 100 MB).  
> Para executar o código e os notebooks, é **obrigatório** baixar o banco pelo link abaixo
> https://drive.google.com/file/d/1B6GccfTa1Wl4fAcV3Afl8K-6cWiSfckP/view?usp=sharing
> e colocá-lo manualmente na pasta `data/`.
